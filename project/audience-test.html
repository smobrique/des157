User Objectives

  1. Users should be able to click through the home screen's slideshow and display three different pictures in the home screen.
  2. Users should be able to click on the link to other people's narratives, and an overlay will appear.
  3. If users try to send an e-mail in the "Contact" section, an alert will pop up saying that the code is still under construction.

Questions for Users

  1. Was the navigation concise/simple to follow?
  2. Would you change how the narratives are displayed (aside from text showing on the screen)? If so, how?
  3. Did the home page slideshow transition each photo smoothly, or was there a lag between each photo?
